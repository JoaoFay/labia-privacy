Política de Privacidade — LabIA

Última atualização: junho de 2026

1. Sobre o aplicativo

O LabIA é um aplicativo de cunho educativo que permite ao usuário inserir resultados de exames laboratoriais e receber interpretações geradas por inteligência artificial. O app não tem finalidade clínica ou diagnóstica e não substitui a avaliação de um profissional de saúde.

2. Dados coletados

O LabIA não coleta, armazena nem transmite dados pessoais identificáveis (nome, CPF, e-mail, telefone, localização, identificadores de dispositivo). Os valores de exames laboratoriais inseridos pelo usuário são armazenados exclusivamente no dispositivo local do próprio usuário.

Além do armazenamento local, os valores numéricos dos exames (sem nenhuma informação de identificação pessoal) são enviados ao backend do LabIA e, em seguida, encaminhados a um serviço de inteligência artificial terceirizado para gerar a interpretação educativa. Essa é a única transmissão de dados do app para servidores remotos.

3. Uso da inteligência artificial

Para gerar interpretações educativas, os valores numéricos dos exames são enviados ao serviço de IA da OpenAI (modelo gpt-4o-mini ou equivalente). Apenas os valores numéricos e o contexto clínico necessário são transmitidos; nenhum dado de identificação pessoal (nome, CPF, e-mail, localização ou qualquer outro identificador) é enviado nesse processo.

Desde 1° de março de 2023, a OpenAI não utiliza dados enviados via sua API para treinar ou melhorar seus modelos de linguagem, a menos que o usuário opte explicitamente por compartilhar seus dados. Isso significa que os valores dos seus exames não são usados pela OpenAI para fins de treinamento.

4. Retenção de dados

Os valores de exames ficam armazenados no seu dispositivo enquanto você mantiver o aplicativo instalado. Você pode apagá-los a qualquer momento pelas configurações do app.

Os valores numéricos enviados à OpenAI são processados em tempo real para gerar a interpretação e não são armazenados de forma permanente pela OpenAI. Consulte a política de privacidade da OpenAI para detalhes sobre a retenção de dados na plataforma deles.

5. Dados não coletados

O LabIA não coleta: nome, e-mail, telefone, localização geográfica, identificadores de dispositivo, dados de uso, analytics, nem qualquer informação que permita identificar o usuário.

6. Crianças

O aplicativo não é destinado a menores de 18 anos e não coleta intencionalmente dados de crianças.

7. Alterações nesta política

Eventuais atualizações serão publicadas nesta página com a data de revisão. O uso continuado do aplicativo após alterações implica concordância com a política vigente.
